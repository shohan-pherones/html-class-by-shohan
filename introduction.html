What is HTML? * HTML stands for Hyper Text Markup Language * HTML is the standard markup language for creating Web pages * HTML describes the structure of a Web page * HTML consists of a series of elements * HTML elements tell the browser how to display the content What is an HTML Element? * An HTML element is defined by a start tag, some content, and an end tag: Content goes here... HTML History * 1989 Tim Berners-Lee invented WWW * 1991 Tim Berners-Lee invented HTML
